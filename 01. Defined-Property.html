物件屬性延伸章節：屬性的特徵

Object.defineProperty(物件, 屬性, 參數)

屬性，調整屬性的特徵 :

  • 1. value : 值
  • 2. writable : 可否寫入 -> 'use strict' 設定 嚴謹模式 ( 跳出錯誤 )
  • 3. configurable : 可否被刪除
  • 4. enumerable : 可否被列舉 -> 禁止寫入限制於 'x' 屬性, 但可對裡面值做額外設定 ( 淺層保護 )
